# First_Week Problem : 문자열 덧셈 계산기
***

## 기능 구현
- 커스텀 구분자 추가 ("//"와 "\n" 사이에 위치하는 문자를 커스텀 구분자로 사용)
- 입력된 문자열 구분자로 자르기
- 구분자로 자른 값 더하기

## 예외 사항
1. 아무 입력이 없는 경우 또는 입력된 문자열에 숫자가 없는 경우
   - 결과 : 0 출력 (덧셈할 숫자가 없기 때문에)
2. 음수값이 포함된 경우
   - IllegalArgumentException을 발생시킨 후 애플리케이션은 종료 (구현하는 애플리케이션은 덧셈 계산기 입니다.)
3. 숫자에 문자가 포함된 경우
   - IllegalArgumentException을 발생시킨 후 애플리케이션은 종료(사용자가 잘못된 입력을 한 경우 입니다.)

## 고려한 점
1. 공백은 커스텀 구분자로 추가되지 않는다.
2. 입력에 공백이 들어와도 상관이 없다.
3. 커스텀 구분자는 두개의 문자를 합쳐서 사용할 수 없다.
   - ex) //*!\n의 경우
4. 숫자는 구분자가 될 수 없다.
5. 실수덧셈도 가능하다.
6. 커스텀 구분자 생성 구문이 입력 문자열 중간에 나와도 이전에 사용할 수 있게 한다.

## 구현 방식
- 입력 문자열을 받으면 해당 문자열에 공백을 제거
- 공백제거된 문자열에 문자가 있는지와 숫자가 있는지 검사하고 없다면 예외처리
- 기본 구분자 리스트 생성("," , ":")
- 입력 문자열에 커스텀 구분자 추가하는 부분이 있으면 기본 구분자 리스트에 추가 후 "//문자\n"문자열에서 제거
- 커스텀 구분자 추가 부분이 없어진 문자열 구분자 리스트에 있는 문자로 구분 (구분된 문자열을 list 자료구조에 저장)
- 구분된 문자열에 숫자가 아닌 문자("."제외한)이 있는경우 예외처리(.은 실수 표현된 값들을 위해 제외 함)
- .의 위치가 맨 앞 or 맨 뒤인 경우와 2개 이상 연속된 경우 잘못된 입력 으로 예외처리
- 구분된 문자열 Double형으로 변환 후 누적 합 출력
